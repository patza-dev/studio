# **App Name**: พร้อมพัฒน์

## Core Features:

- Candidate Profile: Showcase the candidate's bio, photo, and platform summary.
- Team Showcase: Display the candidate's 15-member team with individual profiles.
- Policy Explorer: Provide in-depth descriptions of the candidate's policies and vision, using clear and concise language.
- Q&A Forum: Enable users to submit questions and provide feedback directly to the candidate. Allow candidate to provide responses to frequently asked question
- AI-Powered Policy Summarizer: Uses generative AI tool to automatically create short, easily digestible summaries of the candidate's policies.

## Style Guidelines:

- Primary color: Vibrant pink (#FF69B4) for energy and visibility.
- Background color: Soft, desaturated pink (#F8BBD0).
- Accent color: Electric blue (#7DF9FF) to create contrast and highlight important actions.
- Body and headline font: 'Poppins' sans-serif font, suitable for a contemporary, precise feel. Note: currently only Google Fonts are supported.
- Use modern, minimalist icons to represent policies and team roles.
- Mobile-first, single-column layout for easy navigation and content consumption.
- Subtle transitions and animations for a smooth user experience.
